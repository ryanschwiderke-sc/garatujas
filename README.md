HTML
O HTML é a linguagem de estrutura de uma pagina web. Com ele, criamos os elementos base do site: titulos, subtitulos, paragrafos e as "caixas" (divs e containers) que organiza o conteúdo. Também é no HTML que linkamos o CSS e o JavaScript a página. Cada elemento pode receber uma classe, que depois é referenciado no CSS pra ser estilizada

TypeScript
O TypeScript é tipo uma versão do JavaScript, só que um pouco mais organizada. Ele serve pra ajudar a gente a evitar erro no código, porque dá pra definir o tipo das coisas, tipo número, texto, essas coisas. Assim fica mais facil de entender o que cada parte do código faz

Com TypeScript, as variáveis tem tipo definido, as funções pode dizer o que vai receber e o que vai retornar, e os objetos segue um padrão mais certinho. No começo pode parecer meio complicado, mas ajuda bastante quando o projeto vai ficando maior

CSS
O CSS é a linguagem de estilização responsável pela aparência e organização visual do site. Com ele é possivel centralizar elementos, definir tamanhos e posição, alterar cores de fundo e de texto, ajustar tipografia e torna o layout responsivo para diferente dispositivos (celular, computador, notebook, etc.). Para usar o CSS, é necessario ter o HTML como base — é ele quem cria as "caixas" que o CSS vai estilizar — e linkar ele no arquivo HTML para que a estilização seja aplicada correto
